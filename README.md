# Sistema de Previsão de valor imobiliário

Projeto de Machine Learning para prever a variação dos valores imobiliário de Boston, auxiliando corretores e possiveis compradores e realizarem as suas escolhas.

## Tecnologias Utilizadas

- Pandas
- Numpy
- Matplotlib / Seaborn
- Scikit-learn
- Jupyter Notebook
- CSV

## Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/aSHUA0/Boston-Housing.git
```

2. Navegue até o diretório do projeto:

```bash
cd Boston-Housing
```

3. Instale as dependências:

```bash
pip install -r requirements.txt
```

## Modelagem e Treinamento

Os dados foram pré-processados e utilizados para treinar modelos de classificação, como:

- Random Forest
- Logistic Regression

A avaliação foi feita com métricas como Acurácia, Precisão, Recall e F1-Score.
